Model Based Sentiment classification¶

Model based scoring/classification is a supervised classification suited for sentiment or multi class classification. The idea is to use a labelled training set to learn the associations and score unknown text with associations from the model. Once the model is learnt, tested, validated and accepted, the scoring is real time and can be tested against a labelled/unlabelled data set. In this session we will work with both labelled data sets to implement the naive Bayesian based classifications.
